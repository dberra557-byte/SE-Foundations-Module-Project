# Python To-Do List Application

## Overview
A command-line To-Do List built with Python that allows users to add, view, delete tasks, and quit the program.

## Features
- Tasks stored in a Python list
- Organized using functions
- Input validation
- Error handling with try/except/finally
- Continuous loop until user exits

## How to Run
1. Ensure Python is installed.
2. Open terminal in the project folder.
3. Run:

python todo_app.py
